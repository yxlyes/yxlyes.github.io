使用Hexo搭建博客

若相关安装步骤卡住不动，请考虑网络的原因。

  1. 安装Node.js和npm

    • 首先，你需要在你的计算机上安装 Node.js 和 npm。可以访问 Node.js 的官方网站下载并安装适合你操作系统的版本。
    • 安装完成后，可以在命令行中运行 node -v 和 npm -v 来检查 Node.js 和 npm 是否正确安装。

  2. 安装git

    • Hexo 需要使用 Git 来部署你的博客。你可以从 Git 的官方网站下载并安装。
    • 安装完成后，可以在命令行中运行 git --version 来确认 Git 已正确安装。

  3. 安装Hexo

    • 在命令行中运行以下命令来全局安装 Hexo：npm install -g hexo-cli
